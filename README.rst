
=======================
Search
=======================

This is a Sample Search project.


Requirement
===========

:Python: 3.7.0
:Django: 2.2.5
:django-bootstrap4: 1.0.1

Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/list.git

2. migrate model::

    python manage.py migrate

3. migrate model::

    python manage.py createsuperuser

4. start server::

    python manage.py runserver
